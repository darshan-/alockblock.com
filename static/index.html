A Lock Block

Android's Lock Screen Blocker

A Lock Block is an Android app that allows you to temporarily disable your device's lock screen. You can do this manually at any time (Pro: optionally from the notification), and you can optionally (Pro) have it done automatically when the device is plugged in.

The app is free, which allows you to manually disable and re-enable the lock screen. A very small fee unlocks several convenient features if desired.
